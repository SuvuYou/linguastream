## App description

A personal language learning web app built around consuming content and developing vocabulary linked to the said content
Main features include:

- Having access to public Jellyfin library and ability to upload your own files or YouTube URLs
- Utilizing double subtitles overlaid on the video
- Checking contextual translations of each word
- Saving words to flashcard decks, and reviewing them with spaced repetition
- A Playphrase-style search that lets you find any word or phrase across their entire library and jump to that moment in the video
